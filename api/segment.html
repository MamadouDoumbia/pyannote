Segment¶

class pyannote.Segment(start=0.0, end=0.0)¶

A segment is uniquely defined by its start and end times (in seconds):

>>> from pyannote import Segment
>>> segment = Segment(start=13., end=37)
>>> print segment
[13 --> 37]
