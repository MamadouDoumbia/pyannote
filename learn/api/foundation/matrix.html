Label-indexed matrix¶

class pyannote.base.matrix.Cooccurrence(I, J)[source]¶

Cooccurrence matrix between two annotations

Parameters :I, J : Annotation
Returns :matrix : Cooccurrence

Examples

>>> M = Cooccurrence(A, B)

Get total confusion duration (in seconds) between id_A and id_B:

>>> confusion = M[id_A, id_B]

Get confusion dictionary for id_A:

>>> confusions = M(id_A)
